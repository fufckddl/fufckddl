## 🚀 About Me  
어릴 때부터 새로운 기술이나 구조를 보면 **직접 분해하고 재구성하는 것**에 흥미를 느껴왔습니다.  
그 호기심은 자연스럽게 개발로 이어졌고, 다양한 프로젝트를 통해 다음과 같은 역량을 쌓았습니다:

- 문제를 구조적으로 해석하고, 기술적 해결책을 빠르게 도출하는 사고 방식  
- 모바일 앱 개발부터 AI 모델링, 실시간 음성 처리, 서버 아키텍처까지 확장된 기술 경험  
- 사용자의 흐름을 고려한 **엔드-투-엔드 제품 구현 능력**

현재는 **모바일 기반 AI 서비스**와 **실시간 음성 인식/요약 플랫폼**, **감정 기반 개인화 앱** 등  
다양한 실사용 서비스를 설계하고 구축하고 있습니다.
